# Лабораторная работа №1: Конвертер ставок и вероятностей

Программа представляет собой графическое приложение на JavaFX для взаимной конвертации различных форматов букмекерских котировок (коэффициентов) и расчетной вероятности исходов спортивных событий.

## Функционал
* Конвертация различных форматов ставок
* Поддержка ввода значений в виде десятичных дробей, процентов и обыкновенных дробей
* Автоматический расчет эквивалентных значений

## Стек технологий
* Язык программирования: Java 24/26
* Графическая библиотека: JavaFX 17 (Controls, FXML)
* Система сборки: Maven

## Особенности реализации
* Интерфейс выполнен в современном темном стиле с огненно-оранжевыми неоновыми градиентами
* Реализован отказоустойчивый класс-лаунчер для обхода модульных ограничений JavaFX на новых версиях JDK
